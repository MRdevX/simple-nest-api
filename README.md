## Description

A Simple Book Management API

### Technologies implemented:

-   [TypeOrm](https://typeorm.io/) (ORM) + [PostgreSQL](https://www.postgresql.org/)
-   [JWT](https://jwt.io/)
-   [Swagger](https://swagger.io/)

## Prerequisites

-   [Node.js](https://nodejs.org/) (>= 10.8.0)
-   [npm](https://www.npmjs.com/) (>= 6.5.0)

## Setting up the database

PostgreSQL database connection options are shown in the following table:

| Option   | Development |
| -------- | ----------- |
| Host     | localhost   |
| Port     | 5432        |
| Username | postgres    |
| Password | postgres    |
| Database | book-db     |

## Getting started

```bash
# 1. Clone the repository
git clone git@github.com:MRdevX/simple-nest-api.git my-new-project

# 2. Enter your newly-cloned folder.
cd my-new-project

```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Other commands

```bash
# formatting code
$ npm run format

# run linter
$ npm run lint

```
