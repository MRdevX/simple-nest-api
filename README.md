## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Getting started

```bash
# 1. Clone the repository
git clone git@github.com:MRdevX/simple-nest-api.git my-new-project

# 2. Enter your newly-cloned folder.
cd my-new-project

```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
